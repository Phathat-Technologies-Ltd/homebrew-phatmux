# Homebrew Tap for phatmux

This is the official Homebrew tap for [phatmux](https://github.com/Phathat-Technologies-Ltd/phatmux), a Ghostty-based macOS terminal with vertical tabs for AI coding agents.

## Installation

```bash
brew tap Phathat-Technologies-Ltd/phatmux
brew install --cask phatmux
```

## Updating

```bash
brew upgrade --cask phatmux
```

## Uninstalling

```bash
brew uninstall --cask phatmux
brew untap Phathat-Technologies-Ltd/phatmux
```
